📋 Sobre o Projeto
O ReFood é um aplicativo mobile desenvolvido para ajudar pessoas a controlarem o estoque de alimentos em casa, recebendo alertas antes do vencimento, visualizando sugestões de receitas e acompanhando relatórios de desperdício.
Principais funcionalidades:
🥦 Cadastro e gestão de alimentos com data de validade
🔔 Notificações automáticas antes do vencimento
🍽️ Sugestões de receitas com os ingredientes disponíveis
🛒 Lista de compras compartilhável
📊 Relatório de desperdício com gráficos e economia gerada
👤 Autenticação completa com cadastro, login e recuperação de senha
